How It Works

From domain list
to risk intelligence.

Lancelot takes a list of suspect domains and returns scored, auditable risk assessments with full signal breakdowns.

01

Submit Domains

Upload via API or dashboard. Seed from a file or add individual domains. Lancelot queues them for async processing.

02

Distributed Crawl

Workers claim domains from the task queue and fetch pages. Fault-tolerant by design — failed crawls are re-queued automatically.

03

Extract Signals

Each page is analysed for MFA markers: ad density, script injection, redirect chains, content quality, and network fingerprints.

04

Score Deterministically

Signals are aggregated with weighted scoring. The result is a domain-level risk score with a full breakdown of contributing factors.

05

Monitor & Act

View results in the dashboard or pull via API. Track score drift over time. Flag high-risk domains before they enter your ad supply chain.

FAQ

Common questions.

How is the risk score calculated?
Lancelot extracts page-level signals — script counts, ad network markers, iframe density, link ratios, content quality — and aggregates them with weighted scoring. The formula is deterministic: same signals, same score. Every factor is visible in the signal breakdown.
What signals does Lancelot extract?
Script and iframe counts, Google Ad Manager and AdSense markers, header bidding presence, ad-related keyword density, link-to-content ratios, word count, and redirect chain analysis. More signals are added as fraud patterns evolve.
Can I use the API without the dashboard?
Yes. The REST API is fully independent. Submit domains, retrieve scores, and get signal breakdowns programmatically. The dashboard is optional — use whichever interface fits your workflow.
What's the crawl latency?
Depends on queue depth and domain count. Typical single-domain crawl completes in under 30 seconds. Batch imports are processed asynchronously — webhook notifications fire on completion.
How does this compare to existing ad verification tools?
Most verification tools are black boxes — you get a verdict but can't see the methodology. Lancelot is fully transparent: every signal is visible, every score is reproducible, and the codebase is auditable. You trust the data because you can verify it.
